# 計算機（Calculator）詳細設計書

## 1. 概要

基本的な四則演算（加算・減算・乗算・除算）を提供する汎用計算モジュール。  
他のモジュールや外部スクリプトから `import` して使用することを想定している。

## 2. モジュール情報

| 項目 | 内容 |
|------|------|
| モジュール名 | `calculator` |
| ファイルパス | `src/calculator.py` |
| 対応 Python | 3.10 以上 |
| 外部依存 | なし（標準ライブラリのみ） |

## 3. 関数仕様

### 3.1 `add(a, b)`

| 項目 | 内容 |
|------|------|
| 概要 | 2 つの数値を加算して返す |
| 引数 `a` | `float` — 被加数 |
| 引数 `b` | `float` — 加数 |
| 戻り値 | `float` — a と b の和 |
| 例外 | なし |

```python
add(3, 5)      # => 8.0
add(-1, 1)     # => 0.0
add(0.1, 0.2)  # => 0.30000000000000004
```

### 3.2 `subtract(a, b)`

| 項目 | 内容 |
|------|------|
| 概要 | a から b を減算して返す |
| 引数 `a` | `float` — 被減数 |
| 引数 `b` | `float` — 減数 |
| 戻り値 | `float` — a と b の差 |
| 例外 | なし |

```python
subtract(10, 3)  # => 7.0
subtract(0, 5)   # => -5.0
```

### 3.3 `multiply(a, b)`

| 項目 | 内容 |
|------|------|
| 概要 | 2 つの数値を乗算して返す |
| 引数 `a` | `float` — 被乗数 |
| 引数 `b` | `float` — 乗数 |
| 戻り値 | `float` — a と b の積 |
| 例外 | なし |

```python
multiply(3, 4)    # => 12.0
multiply(-2, 5)   # => -10.0
multiply(0, 100)  # => 0.0
```

### 3.4 `divide(a, b)`

| 項目 | 内容 |
|------|------|
| 概要 | a を b で除算して返す |
| 引数 `a` | `float` — 被除数 |
| 引数 `b` | `float` — 除数 |
| 戻り値 | `float` — a を b で割った商 |
| 例外 | `ValueError` — b が 0 のとき（メッセージ: `"0による除算はできません"`） |

```python
divide(10, 2)  # => 5.0
divide(7, 2)   # => 3.5
divide(1, 0)   # => ValueError: "0による除算はできません"
```

## 4. エラー処理方針

- 引数の型チェックは実施しない（呼び出し元の責任とする）
- ゼロ除算のみ `ValueError` を送出する

## 5. 依存関係

外部ライブラリへの依存なし。標準ライブラリのみ使用。
